Introduction to WAMP and LAMP

1. Introduction to WAMP

WAMP stands for:

  • W – Windows Operating System
  • A – Apache Web Server
  • M – MySQL Database
  • P – PHP
WAMP is a local server environment used to develop and test web applications on a Windows computer. Apache serves the HTML/PHP pages, MySQL stores data, and PHP handles dynamic content.

2. Introduction to LAMP

LAMP stands for:

  • L – Linux Operating System
  • A – Apache Web Server
  • M – MySQL Database
  • P – PHP
LAMP is similar to WAMP but runs on Linux systems. It is widely used for hosting live websites.

3. Benefits of WAMP/LAMP

  • Local Development: Test websites locally before going live.
  • Dynamic Content Support: Run PHP scripts and use MySQL databases.
  • Easy Installation: Bundles Apache, MySQL, and PHP.
  • Cost-Effective: Free and open-source.
  • Cross-Platform Options: WAMP for Windows, LAMP for Linux.
  • Safe Testing Environment: No risk to live servers.

4. Create a Simple "Hello World" Webpage

Steps to create a static HTML page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Hello World Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Welcome to my first webpage using WAMP or LAMP.</p>
</body>
</html>

Save as index.html in WAMP www folder and access via: http://localhost/mywebsite/index.html

5. PHP "Hello World" Example

Steps to create a dynamic PHP page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>PHP Hello World</title>
</head>
<body>
    <h1>
        <?php
            echo "Hello World using PHP!";
        ?>
    </h1>
    <p>Welcome to my first dynamic webpage using WAMP or LAMP.</p>
</body>
</html>

Save as index.php and access via: http://localhost/mywebsite/index.php

Summary

WAMP is ideal for Windows local development, LAMP for Linux servers. Both allow testing static HTML and dynamic PHP pages. Benefits include safe testing, cost-effectiveness, dynamic content support, and easy installation. Creating a simple "Hello World" page demonstrates basic HTML and PHP usage.
