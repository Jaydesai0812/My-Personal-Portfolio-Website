Guide: Installing WAMP and Deploying a Multi-Page Website

Step 1: Install WAMP on Your Computer

WAMP (Windows, Apache, MySQL, PHP) is a local server environment that lets you test websites on your computer. To install WAMP:

  • Download WAMP from https://www.wampserver.com/
  • Install with default settings.
  • Start WAMP. A green icon in the system tray indicates the server is running correctly.
  • Your local server URL is usually: http://localhost/

Step 2: Place Your Multi-Page Website in WAMP

1. Open the WAMP installation folder, usually C:\wamp64\www\.
2. Create a folder for your project, e.g., mywebsite.
3. Place all HTML, CSS, JS, and media files inside.
Example structure: 

C:\wamp64\www\mywebsite\
    index.html
    about.html
    contact.html
    css/
        style.css
    js/
        script.js
    images/
        logo.png

4. Test locally at http://localhost/mywebsite/index.html

Step 3: Test Your Website Locally

- Open each HTML file in your browser using its URL.
- Ensure all links, CSS, JS, and images work correctly.

Step 4: Upload Your Website Using FTP

FTP (File Transfer Protocol) allows uploading to a live server.

  • Install an FTP client (FileZilla).
  • Get FTP credentials from your hosting provider.
  • Connect using FileZilla (host, username, password, port 21).
  • Local panel: navigate to your project folder.
  • Remote panel: navigate to /public_html/.
  • Drag and drop all files from local to remote site.
Visit your live website at http://www.yourdomain.com/index.html

Step 5: Example of Correct File Linking

Use relative paths for CSS, JS, and images so they work locally and online. 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Website</title>
    <link rel="stylesheet" href="css/style.css">
    <script src="js/script.js" defer></script>
</head>
<body>
<nav>
    <a href="index.html">Home</a> |
    <a href="about.html">About</a> |
    <a href="contact.html">Contact</a>
</nav>
<h1>Welcome to My Website</h1>
</body>
</html>

Step 6: Common Issues and Fixes

  • Page not loading: Check file paths.
  • CSS/JS not applied: Verify file links.
  • Images not showing: Match file names exactly.
  • FTP upload fails: Check permissions and credentials.

Step 7: Summary

Steps to deploy a multi-page website:

  1. Install WAMP and place files in www folder.
  2. Test website locally.
  3. Ensure relative paths are correct.
  4. Upload files via FTP to /public_html/.
  5. Verify live website.
